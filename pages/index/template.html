Dashboard demo

This dashboard receives data from the Binance cryptocurrency exchange in real-time using websockets.
You can add, remove, move and resize widgets. User data is stored in Firebase.

Used technologies:

  • TypeScript
  • Nuxt 3
  • Vue 3
  • Pinia
  • Less
Check out the source code at https://github.com/stervets/binance-ts
